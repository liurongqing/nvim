这是一份 nvim 配置


使用了 Packer.nvim 包管理，使用了下列插件

- nvim-tree



1. 准备工作
  1. 环境变量
    `vim ~/.zshrc`
    ```shell
    alias vim="nvim"
    alias vi="nvim"
    alias v="nvim"
    alias n="nvim"
    ```
  1. 安装 tmux
  1. 安装 nvim
  1. 安装 字体
  1. packer.nvim 安装
  1. 克隆整个配置文件
  ```shell
  git clone https://github.com/liurongqing/nvim.git
  ```
  1. packer.nvim 管理的插件安装
